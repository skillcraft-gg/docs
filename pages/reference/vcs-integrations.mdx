# VCS Integrations

Skillcraft integrates with version control platforms to capture repository-level evidence and keep claims in sync with real commits.

## Integration status

| Integration | Status | Notes |
|---|---|---|
| [GitHub](./github) | Supported | Current first-class VCS backend |
| GitHub Enterprise Cloud | Coming soon | Private hosting and enterprise policy support pending |
| Bitbucket | Coming soon | Planned after GitHub Enterprise Cloud rollout |
| GitLab | Coming soon | Planned as a future VCS backend |

See `GitHub` for current setup details. Coming soon entries are reserved for future tracking and documentation consistency.
