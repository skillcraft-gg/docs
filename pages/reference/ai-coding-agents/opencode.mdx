# OpenCode

| Capability | Support |
|---|---|
| Status | Supported |
| Captures provenance | Yes |
| Supports skills | Yes |
| Integration type | Repo-local plugin |

## Status

Supported.

## Current support

OpenCode is supported as an evidence-aware coding agent.

## Enable flow

When you run `skillcraft enable --agent opencode`, Skillcraft writes a managed plugin at:

`.opencode/plugins/skillcraft.mjs`

The plugin captures runtime model provenance from OpenCode chat messages and persists it to:

`.git/skillcraft/ai-model-context.json`

## Captured provenance fields

- `agent.provider`
- `model.provider`
- `model.name`

These values are merged into each generated proof.

## Skill support

OpenCode skill usage is captured through the managed plugin by watching OpenCode skill tool execution and recording the matching installed Skillcraft skill ID.

## Notes

- Keep normal git history visible to Skillcraft.
- Run `skillcraft enable --agent opencode`, or let `skillcraft enable` detect OpenCode automatically.
- Multiple agents can be enabled for the same repository.
- Run `skillcraft disable` to remove the generated plugin and provenance state file.
