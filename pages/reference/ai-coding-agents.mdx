# AI Coding Agents

This table tracks which AI coding agent CLIs are supported by Skillcraft, along with coming soon pages for planned entries.

## Integration status

| Agent | Status | Captures Provenance | Supports Skills | Notes |
|---|---|---|---|---|
| [OpenCode](./opencode) | Supported | Yes | Yes | Repo-local managed plugin integration with skill tool capture |
| [Codex CLI](./codex-cli) | Supported | Yes | Yes | Explicit skill capture today via repo-local hooks; implicit capture is pending an upstream Codex hook patch |
| Claude Code | Coming soon | - | - | Planned support for future release |
| Gemini CLI | Coming soon | - | - | Planned support for future release |
| GitHub Copilot CLI | Coming soon | - | - | Planned support for future release |

Skillcraft currently supports OpenCode and Codex CLI. `skillcraft enable` detects available agents automatically and prompts when both are available unless `--agent` is supplied.
