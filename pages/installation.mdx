# Installation

This page documents the concrete setup flow before you start claim workflows.

## 1. Prerequisites

The following needs to be available in `PATH`:

- `node`, `npm` Node.js and npm ([Download](https://nodejs.org/))
- `git` Git ([Download](https://git-scm.com/downloads))
- `gh` GitHub CLI ([Download](https://cli.github.com/)) and authenticated with `gh auth login`
- `opencode` OpenCode ([Download](https://opencode.ai/))

You will also need:

- a [GitHub](https://github.com/) account
- Internet access

## 2. Install the CLI

Use your preferred package manager:

```bash
npm install -g skillcraft
```

```bash
pnpm add -g skillcraft
```

```bash
yarn global add skillcraft
```

```bash
bun add -g skillcraft
```

## 3. Verify the install

```bash
skillcraft --version
```

If the command is not found, restart your shell and confirm your global bin directory is in `PATH`.
