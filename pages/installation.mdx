# Installation

This page documents the concrete setup flow before you start claim workflows.

## 1. Prerequisites

The following needs to be available in `PATH`:

- `node`, `npm` Node.js and npm ([Download](https://nodejs.org/))
- `git` Git ([Download](https://git-scm.com/downloads))
- `gh` GitHub CLI ([Download](https://cli.github.com/)) and authenticated with `gh auth login`
- one or both supported AI coding agents: `opencode` OpenCode ([Download](https://opencode.ai/)) or `codex` Codex CLI ([Install docs](https://developers.openai.com/codex/cli))

You will also need:

- a [GitHub](https://github.com/) account
- Internet access

## 2. Install the CLI

Use your preferred package manager:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tabs.Tab>

```bash

```

  </Tabs.Tab>
  <Tabs.Tab>

  </Tabs.Tab>
  <Tabs.Tab>

  </Tabs.Tab>
  <Tabs.Tab>


  </Tabs.Tab>

## 3. Verify the install

```bash
skillcraft --version
```

```bash
skillcraft doctor
```

If the command is not found, restart your shell and confirm your global bin directory is in `PATH`.

`skillcraft doctor` reports both OpenCode and Codex CLI so you can confirm which agents are available before enabling a repository.
