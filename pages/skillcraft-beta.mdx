# Skillcraft Beta

Skillcraft Beta turns normal AI-assisted development work into verifiable evidence for credentials. Read the [launch post](https://skillcraft.gg/news/introducing-skillcraft-beta/) for the full announcement.

## Current capabilities

- Enable Skillcraft in a repository with `skillcraft enable`.
- Capture evidence from normal git commits while you work.
- Track credential readiness with `skillcraft progress` and `skillcraft progress track <credential-id>`.
- Submit claims and check claim status with `skillcraft claim`.
- Verify local proof references with `skillcraft verify`.
- Use OpenCode as the currently supported AI coding agent integration.
- Capture OpenCode model provenance in generated proof data.
- Run the current documented beta flow in GitHub-hosted repositories.

## Current limitations

- Beta currently requires a GitHub account.
- Claims must come from commits pushed to public GitHub repositories.
- OpenCode is the only AI coding agent currently supported.
- Other AI coding agent pages in the docs are placeholders for planned support, not active integrations.
- GitHub is the only VCS integration currently documented as supported in beta.
