# Getting started

Welcome to Skillcraft docs.

## Install the CLI

```bash
npm install -g skillcraft
```

## Initialize a repo

```bash
cd your-project
skillcraft enable
```

## Track usage

Create commits with meaningful messages and your skill evidence will be captured. When proofs are ready, run:

```bash
skillcraft progress
skillcraft claim skillcraft-gg/your-credential
```
