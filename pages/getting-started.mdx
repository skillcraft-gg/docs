# Getting started

Welcome to Skillcraft. This guide gets you from CLI install to a working repository setup in a few minutes.

## 1. Install the CLI

```bash
npm install -g skillcraft
```

If you use a different Node package manager, see [Installation](installation).

## 2. Add Skillcraft to a project

From a git repository you want to track:

```bash
cd your-project
skillcraft enable
```

You only need to run this once per repository.

## 3. Run work in the repo

Make a normal development commit while you work:

```bash
touch .skillcraft-guide.md
git add .skillcraft-guide.md
git commit -m "chore: trigger skillcraft evidence"
```

Skillcraft records this activity for evidence capture.

## 4. Verify progress

```bash
skillcraft progress
```

That command shows captured evidence for the current repo and what is available to claim.
