# Getting started

Welcome to Skillcraft. This guide gets you from CLI install to a working repository setup in a few minutes.

## 1. Install the CLI

```bash
npm install -g skillcraft
```

For more detailed installation guidance, including prerequisites, see [Installation](installation).

## 2. Add Skillcraft to a project

From a git repository you want to track:

```bash
cd your-project
skillcraft enable
```

You only need to run this once per repository.

## 3. Work towards a credential

See the [Hello World credential page](https://skillcraft.gg/credentials/skillcraft-gg/hello-world/) for more details about this credential.

```bash
skillcraft progress track skillcraft-gg/hello-world
```

Then run your work through your AI coding agent so Skillcraft can capture the necessary evidence. [OpenCode](reference/ai-coding-agents/opencode) is currently supported. For more about current beta capabilities and limitations, see [Skillcraft Beta](skillcraft-beta).

See [Your first credential](tutorials/first-credential) for the full guided walkthrough including evidence capture and progress checks.

## 4. Claim a credential

```bash
skillcraft claim skillcraft-gg/hello-world
```

You can track the status of your claim with:

```bash
skillcraft claim status skillcraft-gg/hello-world
```

Once your claim is verified, your profile will be added to the issued list for this credential:

- [Issued credentials for Hello World](https://skillcraft.gg/credentials/skillcraft-gg/hello-world/issued/)

Your credential will also appear on your profile and be issued at its own URL, for example:

- [Example credential profile](https://skillcraft.gg/credentials/profiles/github/blairhudson/)
- [Example issued credential page](https://skillcraft.gg/credentials/profiles/github/blairhudson/skillcraft-gg/hello-world/)

From there you can [verify it](https://skillcraft.gg/credentials/profiles/github/blairhudson/skillcraft-gg/hello-world/?verify) and [share it on LinkedIn](https://skillcraft.gg/credentials/profiles/github/blairhudson/skillcraft-gg/hello-world/?share).
