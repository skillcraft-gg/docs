# Roadmap

This page tracks public Skillcraft direction from labeled GitHub issues across published repos. It updates as roadmap labels move, and live refresh can pick up newly published metadata without rebuilding docs. [Join the Discussion](https://github.com/orgs/skillcraft-gg/discussions/new/choose) if you want to help shape what comes next.

Roadmap labels:

- `roadmap:now` for active work in flight.
- `roadmap:next` for queued work expected soon.
- `roadmap:later` for longer-horizon work.
- `roadmap:released` for shipped work kept visible in roadmap history.
