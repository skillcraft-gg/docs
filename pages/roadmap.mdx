# Roadmap

This page tracks the public direction for Skillcraft. It will evolve based on Beta feedback from the community. [Join the Discussion](https://github.com/orgs/skillcraft-gg/discussions/new/choose) if you want to help shape what comes next.

## Current focus

- Expand the beta beyond the first public GitHub workflow while keeping verification clear and trustworthy.
- Improve the end-to-end developer experience across the CLI, docs, and `Skillcraft.gg`.
- Make published credentials, skills, and supporting evidence feel more connected and easier to navigate.
