# Roadmap

This page tracks the public direction for Skillcraft Beta. It reflects work already visible in the docs, published placeholder routes, and active planning documents. It is intentionally high level and does not promise release dates.

## Current focus

- Make AI-assisted development work easier to prove with verifiable, git-native evidence.
- Keep the beta flow simple: enable a repo, keep working normally, track progress, and claim credentials when evidence is ready.
- Improve the developer experience around progress tracking, verification, and claim readiness.

## Planned AI coding agent support

OpenCode is supported today. The current docs already reserve stable routes for additional planned agent integrations:

- Claude Code
- Gemini CLI
- GitHub Copilot CLI
- Codex CLI

## Planned VCS integration support

GitHub is the current documented beta path. Additional integration pages are already reserved for:

- GitHub Enterprise Cloud
- GitLab
- Bitbucket

## Evidence and credential roadmap themes

- Richer captured workflow context, including agent and model provenance.
- Better progress evaluation across tracked credentials and repositories.
- Broader credential coverage beyond the first beta workflow.
- Clearer verification and troubleshooting guidance as the system matures.

## Product and docs expansion

- Expand public documentation for planned integrations as they move from placeholder status to supported workflows.
- Continue improving credentials, profile pages, and other discovery surfaces across the Skillcraft ecosystem.
- Publish more examples that show what strong verifiable evidence looks like in real projects.
